# Personal website repo

This site is built with [Eleventy](https://www.11ty.dev/) and deployed to GitHub Pages.

## Build locally

```bash
npm install
npm run build
```

Generated site output is written to `public/`.

## Deployment

A GitHub Actions workflow (`.github/workflows/deploy.yml`) builds and deploys the `public/` directory to GitHub Pages on pushes to `main`.

## Weeknotes URLs and redirects

- Individual weeknotes now use a date-and-title URL pattern, e.g. `/weeknotes/2021-04-05/weeknotes-1/`, derived from the `date` and `title` front matter.
- Legacy URLs based on season/episode (e.g. `/weeknotes/s01e01/`) are preserved via static HTML redirect pages generated by `src/content/weeknotes-redirects.njk`, which immediately forward to the new URLs.
